# TOP Project - Knights Travails

Find the shortest path between two coordinates in chess board for a knight piece, using recursion.

Example output:

```
 > knightMoves([3, 3], [4, 3])
  => You made it in 3 moves!  Here's your path:
    [3, 3]
    [1, 2]
    [3, 1]
    [4, 3]
```
